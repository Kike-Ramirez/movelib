MoveLib

A library by Raphael de Courville for the Processing programming environment.
Last update, 05/04/2013.

Add support for the PS Move Motion Controller in Processing. MoveLib is based on Thomas Perl's PSMove API.
The lib was tested on OSX and should also run on Linux. On Windows, bluetooth pairing is still an issue so the OS is considered unsupported until a workaround is found.

Download

Download MoveLib version 0.1b1 (1) in .zip format.

Installation

Unzip and put the extracted MoveLib folder into the libraries folder of your Processing sketches. Reference and examples are included in the MoveLib folder.

Keywords. move, motion, control, hardware, controller, gyroscope, accelerometer, magnetometer, sensor, fusion, IMU, tracking, orientation, quaternion, peripheral, bluetooth
